## Overview

This repository provides the STM32Cube **Sub-GHz Phy** Middleware Library, **common to every** STM32Cube MCU firmware that supports it.

## Compatibility information

Please refer to the **release note** in the firmware repository for the STM32 series you are using. It is **important** to use a **consistent set** of software versions as specified.
